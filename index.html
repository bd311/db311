1960s

Bases de Datos Jerárquicas y de Red

Jerárquicas: Datos organizados en una estructura de árbol. Ejemplo: IBM IMS.

De Red: Datos representados en redes de nodos y enlaces. Ejemplo: IDS.

1970s

Modelo Relacional

Introducción del modelo basado en tablas y SQL. Ejemplo: IBM System R, Oracle Database.

1980s

Bases de Datos Orientadas a Objetos

Datos y procedimientos integrados en objetos. Ejemplo: ObjectDB, Versant.

1990s

Bases de Datos Distribuidas y NoSQL

Distribuidas: Datos en múltiples ubicaciones físicas. Ejemplo: Oracle Parallel Server.

NoSQL: Modelos no relacionales para datos diversos. Ejemplos: MongoDB, Cassandra.

2000s+

Bases de Datos en la Nube y Nuevas Tecnologías

En la Nube: Escalabilidad bajo demanda. Ejemplos: Amazon RDS, Google Cloud SQL.

NewSQL: Consistencia del modelo relacional con escalabilidad NoSQL. Ejemplos: Google Spanner.
